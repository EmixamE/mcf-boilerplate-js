# `@mcf/constants`
Exposes constants used across MCF products.

## Installation
Install it via `npm` or `yarn`:

```sh
npm i @mcf/constants
# or
yarn add @mcf/constants
```

## Usage

### Requiring
```js
import {icms, mcf, msf} from '@mcf/constants';
// or
const {icms, mcf, msf} = require('@mcf/constants');
```
